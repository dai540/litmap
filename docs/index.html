litmap is a Python package for reproducible literature mapping. It is designed around explicit runs, inspectable artifacts, and a clean separation between clustering space and plotting space.

If you are new here, start with a tutorial. If you are evaluating the package itself, read the design notes and public API. If you only need the architecture, the pipeline diagram below is the shortest overview.

Architecture

Pipeline at a glance

A run starts with a corpus definition, creates a canonical paper table, builds embeddings, decides cluster structure in analysis space, and only then creates a 2D display layer for the map.

Quick start

Minimal local workflow

pip install -e .[docs]
mkdocs serve
litmap init-config --output configs/my-run.yaml
litmap show-plan --config configs/default.yaml
