litmap is a Python package for reproducible literature mapping. It keeps one run readable by separating corpus preparation, analysis-space clustering, and display-space projection into explicit artifacts.

The package is for workflows where a map is not enough on its own. You also want labels, summaries, coordinates, diagnostics, and a stable run layout that can be reviewed later.

Architecture

How one run is organized

The package follows a simple pipeline: build one canonical paper table, decide cluster structure in analysis space, and only then project into a separate display layer for reading and sharing.

Quick start

Minimal commands

pip install -e .[docs]
mkdocs serve
litmap init-config --output configs/my-run.yaml
litmap show-plan --config configs/default.yaml
