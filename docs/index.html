litmap is a Python package for reproducible literature mapping. It is built for workflows that need a stable artifact tree, a clear clustering pipeline, and tutorials that end with concrete review outputs rather than screenshots.

The docs are organized so you can decide quickly where to start: tutorials if you want worked examples, API if you want the public interface, design if you want the architecture, and the repository if you want the implementation.

Pipeline

How one run is structured

The package uses a fixed conceptual flow: define a corpus, prepare paper text, build embeddings, cluster in analysis space, and project into a separate 2D display space. The split between analysis space and display space is the main thing to understand before reading the tutorials.

Artifacts

Outputs you should expect

  • labels.csv
    Paper-level cluster labels and membership probabilities.
  • cluster_summary.csv
    Cluster sizes, persistence, and representative papers.
  • coords_2d.csv
    Display-space coordinates used for plotting.
  • map_interactive.html
    Interactive Plotly map with hover details.
